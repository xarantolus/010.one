Music Server

sensibleHub is a self-hosted music management server.

It allows managing your music collection from any device and syncing using external programs. That way, you'll always have the latest files on all your devices.

The server backend is written in Go, the front-end uses normal JavaScript and is mostly styled using the Bulma CSS framework. The server provides live updates video WebSockets, so any changes will be seen on all connected clients.

GitHub repository

Web Archive Server

Collect is a self-hosted server for archiving websites.

Your personal web archive that supports downloading normal websites and videos. When a webpage goes offline or is deleted, you can still view it.

The server backend is written in TypeScript, the front-end is a single-page application written in vanilla JavaScript. It is styled using the UIkit CSS framework. The server provides live updates via WebSockets, which means that any change will be propagated to all connected clients.

Android Customizations

backtap makes the fingerprint sensor more useful.

The Magisk module starts a custom program when booting the Android device. This programs listens for taps on the fingerprint sensor and converts them to clicks on a specific region of the screen.

To write signals to the screen, I needed to know how the Linux multitouch protocol works. I also wrote a blog post with my findings.

GitHub repository Read the blog post

AdBlock Filterlists

filtrite is a fully automated AdBlock filter list generator for Bromite.

The browser Bromite for Android allows custom AdBlock rules in a specified format. This program generates this type of file from an input of online blocklists.

In order to keep the generator up to date, I needed to compile parts of the Chromium project. This now happens every week on GitHub Actions in the fully automated subresource_filter_tools repository.

GitHub repository

Rover

gover is a simple web-controlled rover powered by a Raspberry Pi.

This Raspberry Pi powered rover can be controlled either via a keyboard, an Xbox controller or simply tilting a phone. It avoids frontal collisions using the ultrasonic sensor.
