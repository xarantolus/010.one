Hi, I'm a student from Germany that discovered programming a few years ago. Ever since then I have been fascinated by computers and programming. You might be interested in my GitHub or Stack Overflow profile to see my latest published works. My skills:

  • Web Development

    I really like using Go to build the back-end of web services. Most of them also need a front-end and an API.
    My focus is on a clean user interface that is updated whenever something on the server side changes. This is why I often implement WebSocket endpoints that provide live events and updates.
    In the past, I have also used TypeScript and NodeJS to create web applications.

  • Robotics

    I'm a fan of robotics which is why I built my own rover. It can be controlled using all kinds of devices, e.g. via smartphone, PC or XBox controller. Combining hardware and software is not just very interesting but also really enjoyable.

  • Cross-platform Applications

    Another thing I like is building applications for the Raspberry Pi.
    Examples for these applications are (Chat-)Bots, Servers, and Crawlers.

  • Desktop Applications

    I have also built small desktop applications using C# and WPF. And also in Windows Forms but that's quite old and I don't like it anymore.
    They include an user interface, background tasks and are usually able to access APIs.

  • Other

    Other things I sometimes create include browser extensions, web scrapers, Twitter bots and smaller command-line utilities.
